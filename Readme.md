# frontend

## Proje kurulumu

```
yarn install
```

### Çalıştırmak için

```
yarn serve
```

### Derleyip inşaa etmek için

```
yarn build
```

# backend

## Proje kurulumu

```

yarn install

```

## Prisma ORM ile TypeScript üzerinde Database modellerinizi Generate edin.

```

yarn prisma:migrate

```

## Mock datalar ekleyerek direkt test etmeye başlayın

```

yarn app:seed

```

## Artık hazırsınız, projeyi başlatın.

```

yarn start:dev

```
